spacehatch/oauth

Sign in once — no token to paste.

This variant swaps the personal access token for a GitHub sign-in. If you are already signed in to GitHub and have authorized the app before, the button returns you here silently and launches a Codespace for ….

Step 1 · Sign in

Uses the OAuth authorization-code flow with PKCE. A minimal, stateless function performs the code exchange; the access token then lives only in this tab's memory.

Step 2 · Launch

Open terminal ↗

The terminal opens automatically in a new tab on *.app.github.dev once the codespace is ready — a first cold start takes 1–4 minutes.
