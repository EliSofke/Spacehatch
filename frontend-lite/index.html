spacehatch/lite

One click. No token, no sign-in flow.

This launcher rides your existing GitHub session — the same mechanism VS Code Web uses. It resumes your most recent codespace for the repository, or creates one, entirely on GitHub's side. Nothing to configure, nothing to paste, works on any bare repository.

Manage codespaces ↗

Opens the codespace in VS Code Web — the terminal is one Ctrl+` away (or Menu → Terminal → New Terminal). Quickstart links always open in the web client. If your account's editor preference is JupyterLab, opening a codespace from your codespaces list lands in JupyterLab's terminal instead.

auth: your github.com session · launch: codespaces.new quickstart · zero infrastructure, zero tokens
